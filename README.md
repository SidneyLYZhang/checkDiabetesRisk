# 糖尿病风险测评

因为父母年纪渐大，对他们身体健康的关心甚感不足，我也没有时间好好带他们体检。
偶见家里年长的亲戚也都多有健康问题，尤其血糖问题较多的亲戚也越来越多。
想着至少要有一个更好的方式来让自己爸妈重视一些问题，这只是为了让他们可以自己重视自己的一个途径。
